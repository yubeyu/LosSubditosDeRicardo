El <script> elemento también debe ir al encabezado y debe incluir un srcatributo que contenga la ruta al JavaScript que desea cargar y defer, que básicamente le indica al navegador que cargue el JavaScript al mismo tiempo que el HTML de la página. Esto es útil ya que se asegura de que el HTML se cargue todo antes de que se ejecute JavaScript, de modo que no obtenga errores como resultado de que JavaScript intente acceder a un elemento HTML que aún no existe en la página. En realidad, hay varias formas de manejar la carga de JavaScript en su página, pero esta es la más infalible para usar en los navegadores modernos (para otros, lea las estrategias de carga de scripts ).
